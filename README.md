# Project Managment gh extension

## Requirements

- [Go](https://golang.org/dl/)
- [GitHub CLI](https://cli.github.com/)

## How to develop

```bash
go mod tidy
make dev
```

- Testing repository: https://github.com/Zytera/project-managment-test

```bash
$ git clone https://github.com/Zytera/project-managment-test
$ cd project-managment-test
$ gh project-managment
```
